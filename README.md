# paint-cost-calculator
With this app you can determine how much paint you need for painting the house and how much it costs.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:5173](http://localhost:5173) in your browser.

## Features

-   **Configuration**: Define paint types with coverage and price.
-   **Calculator**: Create rooms, add walls, and assign paints.
-   **Persistence**: All data is saved to your browser's local storage.
-   **Summary**: View total paint required and estimated cost.
